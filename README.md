# 性能带宽计算器

网页数据分析工具，自动计算数据统计信息和 LACU 性能区间。

## 功能特点
- 📊 支持多种数据输入格式
- 🔄 两种计算模式（小好/大好）
- 📈 可视化图表展示
- 📱 手机电脑都能用
- 💾 可下载分析结果

## 快速使用
1. 输入数据（如：4.4, 3.75, 6.23）
2. 点击"计算并绘制图表"
3. 查看 LACU 区间结果
4. 下载图表保存分享

## LACU 区间
- **L**：最优性能（红色）
- **A**：领先性能（橙色）
- **C**：竞争性能（绿色）
- **U**：普通性能（蓝色）
